# 📊 Sales Data Analysis Dashboard

## 📌 Project Overview
This project analyzes sales performance using a transactional sales dataset. The objective of the project is to explore sales trends, product performance, and regional sales distribution through an interactive Power BI dashboard.

The dashboard helps identify key revenue drivers and provides insights that can support data-driven business decisions.

---

## 📂 Dataset
The dataset contains transactional sales information including:

- Order ID
- Product Name
- Product Category
- Sales Amount
- Quantity Sold
- Profit
- Order Date
- Customer Segment
- Region

The data was cleaned and analyzed to uncover important patterns in sales performance.

---

## 📊 Key Performance Indicators (KPIs)

The dashboard tracks several important business metrics:

- **Total Sales**
- **Total Profit**
- **Total Orders**
- **Average Order Value**
- **Total Quantity Sold**

These KPIs help measure overall business performance.

---

## 📈 Dashboard Features

### Sales Performance Analysis
- Total sales by product category
- Monthly sales trends

### Product Performance
- Top performing products by revenue
- Product contribution to total sales

### Regional Sales Insights
- Sales distribution across different regions
- Identification of high-performing regions

### Profit Analysis
- Profit contribution by category
- Profit trends over time

---

## 🛠 Tools & Technologies Used
- **Power BI** – Data visualization and dashboard development  
- **Data Cleaning**  
- **Exploratory Data Analysis**

---

## 🔍 Key Insights
Some important insights from the analysis include:

- A small number of products generate a large portion of total revenue.
- Certain regions contribute significantly more to overall sales.
- Some product categories show higher profitability than others.
- Sales trends show seasonal variations across different months.


---

## 🎯 Project Objective
The objective of this project is to demonstrate data analysis and visualization skills by transforming raw sales data into meaningful insights that help understand business performance.
